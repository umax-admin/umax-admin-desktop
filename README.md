# umax-admin

> 基于 umi4 antd design 开源后台管理系统

## Umax 强大功能

### 微生成器

#### 基本使用

```bash
    umi generate
    # 或者
    umi g
```

#### 直接生成

```bash
umi g page foo

# Write : src/pages/foo.tsx
# Write : src/pages/foo.less
```

#### 组件生成器

```bash
umi g component


```

#### 已目录方式生成页面

```bash
    umi g page bar --dir
```

#### 接口文件生成器

```bash

umi g api

```

#### Mock 生成器

```bash
umi g mock acl
```
